Augmented Application Front End

Augmented front end is a React single page application used to track fitness and nutrition. The end goal for augmented is to have users able to record exercises routines, sleep schedules, meals, etc., connect wearable and smart devices to the application, and access other fitness/nutrition/diet tracking applications, to have one place to collect all self tracking data. Currently, the application is only used to track cardio workouts. Application uses Bulma.css framework, is deployed using NPM to an S3 bucket, and was GoDaddy forward masking that web page at http://augmented.fit.
